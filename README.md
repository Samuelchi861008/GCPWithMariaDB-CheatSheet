# GCP with MariaDB

## Step 1. 建立 Compute engine VM
  * Name
    * 隨意取名
  * Region
    * 選擇 asia-east-1(Taiwan) 與 asia-east1-c，資料中心於彰化，速度比較快
  * Machine type
    * 無特殊需求設定最小即可
  * Boot disk
    * 選擇 Debian GNU/Linux 9(stretch)或其他廠牌Linux皆可，並設定SSD，看需求設定SSD大小，建議15G
  * Identity and API access
    * 選擇預設
